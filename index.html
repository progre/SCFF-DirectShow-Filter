SCFF DirectShow Filter

"ScreenCapture for FFmpeg" DirectShow Filter

Download this project as a .zip file Download this project as a tar.gz file

SCFF directShow FilterはWindows用スクリーンキャプチャプログラム(DirectShowフィルタ)です。ffmpegやWindows Media Encoderの映像入力として使われることを想定しています。

特徴

  • 「より綺麗に」
    • WME/ffmpeg/libav/x264/Flash Media Encoder/xSplitなどのエンコードに適したYUV420Pカラー出力が可能になりました。
  • 「たくさんの種類から拡大縮小方式を選べるように」
    • 拡大縮小機能に高品質な出力で有名なffmpeg/libavのSWScaleを利用しています。
    • これにより10種類の拡大縮小方式を利用可能になりました。
  • 「より軽く、より早く」
    • 内部および出力形式がキャプチャプログラムで一般的なRGB0(32bit)からYUV420P(12bit)になりました。
    • これによりエンコード全体の短時間化（高速化）に貢献しています。
